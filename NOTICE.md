# Notices

Languages: [English](NOTICE.md) | [简体中文](NOTICE.zh-CN.md)

This repository is a multi-license profile registry. Repository tooling,
documentation, schemas, manifests, index files, and profile assets may be
licensed under different terms.

Resource-specific attribution and notices belong next to the resource they
cover:

- `profiles/**/manifest.json` records source, author, license, and
  redistribution permission.
- `profiles/**/LICENSE` contains the resource-local license text when required.
- `profiles/**/NOTICE.md` contains required attribution, provenance notes, and
  modification history for that resource.

No third-party profile assets are currently included in this repository.

When third-party assets are added, this file may include a high-level index of
those notices, but the resource-local manifest and NOTICE file remain the
authoritative location for asset-specific obligations.
