# Notices

语言: [English](NOTICE.md) | [简体中文](NOTICE.zh-CN.md)

本仓库是一个多许可证 profile registry。仓库工具、文档、schema、manifest、index 文件和 profile asset 可能采用不同条款。

资源专属 attribution 和 notice 应放在其覆盖的资源旁边：

- `profiles/**/manifest.json` 记录 source、author、license 和再分发许可。
- `profiles/**/LICENSE` 在需要时包含资源本地的 license text。
- `profiles/**/NOTICE.md` 包含该资源必需的 attribution、来源说明和修改历史。

本仓库当前不包含任何第三方 profile asset。

添加第三方资产后，本文件可以包含这些 notice 的高层索引，但资源本地的 manifest 和 NOTICE 文件仍是资产专属义务的权威位置。
